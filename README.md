## 插件说明

**声明：本插件仅为学习交流使用，请勿用于商业途径。插件基于chrome扩展架构开发，还有很多待完善的地方，请见谅。可能遇到图像与字幕有点对不上，那是智云课堂本身的问题，本插件已经尽力调整了**

### 用途：自动将课堂PPT和讲解字幕导出为PDF文档或者Markdown文件（含图片的压缩包）

### 特点：

#### 1.每页pdf或者markdown有页眉（具体时间），页脚（第几页），以及ppt对应的字幕

#### 2.添加重复图像识别，自动过滤重复ppt

#### 3.导出的PDF文件或者Markdown文件有课程标题和时间，方便查看



## 安装插件

### 你需要准备什么：chrome/edge浏览器

无论是chrome还是edge用户，直接进入[Chrome 应用商店](https://chromewebstore.google.com/)（科学上网）搜索【ZYNotes】，然后获取插件即可。

安装完成后点击右上角扩展程序→小图钉将插件固定即可使用

需要进入智云课堂具体回放页面后再导出
